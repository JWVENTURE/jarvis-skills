## 📊 Color Usage Guidelines

### When Creating Documents

| Document Type | Brand to Use | Primary Color |
|---------------|--------------|---------------|
| JW Ventures Contract | JW Ventures | Olive `#8B9A6A` |
| JW Ventures Invoice | JW Ventures | Olive `#8B9A6A` |
| PickleballBN App UI | PickleballBN | Midnight Blue `#002c72` |
| PickleballBN Website | PickleballBN | Midnight Blue `#002c72` |
| UniPlay Platform | UniPlay | Navy `#0A2240` |
| MeterAI Product | MeterAI | Purple `#4B0082` |
| Covera Document | Covera | Blue `#3498DB` |

### WCAG Contrast Reminder

Always ensure:
- Text on background: minimum 4.5:1 contrast
- Large text: minimum 3:1 contrast
- UI components: minimum 3:1 contrast

### No Color Vibrations

Avoid:
- Bright red on bright blue
- Chartreuse on white
- Cyan on white
- Any bright-on-bright combination

Safe:
- Dark on light
- Light on dark
- Brand colors on their proper backgrounds

---

## 🚨 Common Mistakes to Avoid

1. ❌ Using PickleballBN colors in JW Ventures documents
   → Use JW Ventures Olive Green for contracts/invoices

2. ❌ Using JW Ventures colors in PickleballBN app
   → Use PickleballBN Midnight Blue for app/website

3. ❌ Using wrong brand colors for platform
   → Check which brand the document/product is for

4. ❌ Mixing brand colors in one document
   → Stick to one brand's color scheme

5. ❌ Using deprecated PickleballBN colors
   → Use current palette (Midnight, Chartreuse, Cyan)
